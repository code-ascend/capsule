# capsule

A tool for building portable Linux containers from OCI images. The result is a single ELF binary with a statically
compiled Go runtime inside.

[Documentation (English)](README.en.md)

Инструмент для создания портативных Linux-контейнеров из OCI-образов. Результат — единый ELF-бинарник со статически
собранным Go-рантаймом внутри.
